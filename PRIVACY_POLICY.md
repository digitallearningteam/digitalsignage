# Privacy Policy for EduBuzz Whiteboard

**Last Updated:** November 24, 2025

## Overview

EduBuzz Whiteboard ("the Extension") is a Chrome browser extension designed for educational use by ELC Schools. This privacy policy explains how we handle data when you use our whiteboard extension.

## Information We Collect

### Data Stored Locally

The Extension stores the following data **locally on your device only** using Chrome's Storage API:

- **Whiteboard drawings and strokes** - All pen, eraser, and drawing data you create
- **Educational tool placements** - Positions and configurations of number lines, ten frames, counters, and alphabet rulers
- **Page data** - Multiple whiteboard pages and their contents
- **User preferences** - Tool settings such as brush size, colors, stroke styles, and background preferences

### Data We Do NOT Collect

- **No personal information** - We do not collect names, email addresses, or any personally identifiable information
- **No usage analytics** - We do not track how you use the extension
- **No network transmission** - Your whiteboard data never leaves your local device
- **No cookies** - We do not use cookies or tracking technologies
- **No third-party services** - We do not integrate with or send data to any third-party services

## How We Use Your Data

All data created in the Extension is:

- Stored locally in your Chrome browser's storage
- Used solely to save and restore your whiteboard sessions
- Never transmitted over the network
- Never shared with ELC Schools, developers, or any third parties
- Automatically saved to prevent data loss

## Data Storage and Retention

- All data is stored using Chrome's local Storage API
- Data persists across browser sessions until manually deleted
- You can clear all Extension data at any time by:
  - Using the "Clear Page" button within the Extension
  - Uninstalling the Extension
  - Clearing Chrome extension data via `chrome://extensions/`

## Children's Privacy

The Extension is designed for educational use and may be used by students of all ages. We do not knowingly collect any personal information from children or adults. All whiteboard data remains local to the user's device.

## Permissions

The Extension requests the following Chrome permissions:

- **Storage** - To save your whiteboard data locally on your device
- **No other permissions** - We do not request access to your browsing history, personal data, or any other browser features

## Data Security

- All data is stored locally using Chrome's secure Storage API
- No data is transmitted over networks
- No server-side storage or cloud synchronization
- Data is protected by Chrome's built-in security measures

## Updates to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this document. Continued use of the Extension after changes constitutes acceptance of the updated policy.

## Data Deletion

You have complete control over your data and can delete it at any time:

1. **Clear individual pages** - Use the "Clear Page" button in the Extension
2. **Clear all data** - Uninstall the Extension from Chrome
3. **Chrome storage management** - Go to `chrome://extensions/`, find EduBuzz Whiteboard, and clear site data

## Export and Portability

The Extension provides export functionality:

- Export whiteboard pages as PNG images
- Export complete whiteboard data as JSON files
- Import previously exported JSON files

These features give you full control and ownership of your work.

This Extension complies with:

- Children's Online Privacy Protection Act (COPPA) - We do not collect personal information from children

## Your Rights

As a user, you have the right to:

- Access all your locally stored data through the Extension interface
- Delete your data at any time
- Export your data in portable formats (PNG, JSON)
- Use the Extension without providing any personal information
- Uninstall the Extension and have all local data removed

## Third-Party Links

The Extension does not contain links to third-party websites or services.

If we add features that change how data is handled (such as cloud sync or analytics), we will:
